# dataset_final

Итоговый датасет для обучения детектора заболеваний пшеницы.

## Состав

- **Classes**: 9 — Недостаток P2O5, Листовая (бурая) ржавчина, Мучнистая роса, Пиренофороз, Фузариоз, Корневая гниль, Септориоз, Недостаток N, Повреждение заморозками.
- **train**: 10 855 изображений / 33 967 аннотаций
- **val**: 889 изображений / 3 456 аннотаций
- **test**: 445 изображений / 1 474 аннотации

## Как собран (пайплайн)

1. **Task 02**: экспорт Label Studio → YOLO (`convert_ls_to_yolo.py`). Фильтр: класс остаётся, только если на нём ≥ 100 снимков (`MIN_IMAGES=100`). Это даёт 9 рабочих классов; 9 редких патологий, присутствующих в JSON только как со-аннотации (1–8 снимков), отброшены.
2. **Task 04**: классическая аугментация (Albumentations) — каждое train-изображение × 3 (оригинал + 2 аугментации: гео + фото + полевые условия + шум).
3. **Task 05**: oversampling классов ниже 80% медианы через агрессивные аугментации (Elastic, GridDistortion). Закрывает **70% дефицита** до цели.
4. **Task 06**: Stable Diffusion img2img (SD-1.5) — 450 синтетических изображений для 4 недопредставленных классов. Закрывает оставшиеся **30% дефицита**. Метки наследуются от seed-изображений (strength=0.40 сохраняет композицию).

## Ограничения

- **Метки синтетических изображений унаследованы** от seed-изображений. При `strength=0.40` композиция сохраняется с точностью до ~60% пикселей — это валидное приближение, но не точное. Для high-precision обучения требуется ручная переразметка bbox на сгенерированных снимках или использование inpainting с маской.
- **SD-1.5 не дообучалась** на пшеничных снимках — промпты интерпретируются «стоково», без понимания конкретных сортов и фаз вегетации. LoRA/Dreambooth на 100–500 реальных примерах дало бы прирост реализма.
- **Исключены 9 редких патологий** (Повреждение гербицидами, Чернь колоса, Стеблевая ржавчина, Пыльная головня, Спорынья, Склеротиниоз, Вирус полосатой мозаики, Тёмнобурая пятнистость, Твёрдая головня) — у них <100 снимков в датасете, обучать невозможно.
