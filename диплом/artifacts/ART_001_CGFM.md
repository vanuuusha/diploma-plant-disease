# ART-001: CGFM — Context-Gated Feature Modulation

> **Тип:** Решение по новаторству / ключевой алгоритм дипломной работы
> **Дата:** 2026-04-14
> **Статус:** Принято
> **Контекст:** Обсуждение технического новаторства для дипломной работы

---

## Суть в одном абзаце

CGFM (Context-Gated Feature Modulation) — предлагаемый модуль, в котором отдельный лёгкий контекстный энкодер анализирует всё изображение целиком и генерирует параметры модуляции (γ, β), перенастраивающие карты признаков внутри neck детектора YOLOv12 **до** предсказания боксов. Механизм основан на FiLM (Feature-wise Linear Modulation, Perez et al., 2018), но перенесён из области VQA в neck объектного детектора для задачи диагностики заболеваний пшеницы — чего ранее не делалось.

---

## Проблема

В двухветвевой архитектуре из тезисов контекст подключается **после** детекции (late fusion) — в классификаторе. Детектор при этом «слеп» к контексту: не знает масштаб съёмки, условия освещения, фазу вегетации. Из-за этого:
- Пропускает мелкие симптомы на фото с расстояния (Recall падает)
- Путает тени/фон с болезнями (Precision падает)
- Не различает визуально похожие болезни (пиренофороз vs септориоз)

---

## Математика

```
c = E_ctx(x)                        — контекстный энкодер → вектор c ∈ R^d
γ_l = σ(W_γ^l · c + b_γ^l)         — scaling ∈ R^C (сигмоида, [0,1])
β_l = W_β^l · c + b_β^l            — shifting ∈ R^C
F'_l = γ_l ⊙ F_l + β_l            — модулированные признаки
```

Применяется на каждом уровне FPN/PAN neck (P3, P4, P5). Обучается end-to-end через loss детектора.

## Что контекстный энкодер выучивает (имплицитно)

Явных меток нет — обучается end-to-end. Но через визуализацию γ-значений можно показать, что он выучил:
- **Масштаб съёмки** → модулирует чувствительность к размеру объектов
- **Освещение** → подавляет каналы, реагирующие на блики/тени
- **Фаза вегетации** → усиливает каналы, релевантные для текущей стадии
- **Сложность фона** → подавляет фоновые false positives

## В чём новаторство

**Ключевое отличие от существующих методов внимания:**

| Метод | Источник сигнала | Тип модуляции | Проблема |
|-------|-----------------|--------------|----------|
| SE-Net (2018) | Сами признаки (global avg pool) | Каналы | Замкнутый круг — рекалибрует сам себя |
| CBAM (2018) | Сами признаки (channel + spatial) | Каналы + пространство | То же — нет внешнего контекста |
| ECA-Net (2020) | Сами признаки (1D conv) | Каналы | Лёгкий, но самореферентный |
| FiLM (2018) | Внешний сигнал (текст вопроса) | Каналы (γ, β) | Применялся в VQA, **не в детекции** |
| YOLO-DC (2025) | Deformable conv | Пространство | «Контекст» = мультимасштаб, не внешний |
| **CGFM (наш)** | **Отдельный энкодер сцены** | Каналы neck (γ, β) | — |

SE/CBAM берут сигнал из тех же признаков, которые модулируют. CGFM вводит **внешний** сигнал от отдельного энкодера, видящего всю сцену. FiLM-подобная модуляция в neck детектора для аграрных задач — не делалось.

## Конфигурации для сравнения (аблация)

### Основная таблица: 6 конфигураций

| # | Конфигурация | Описание | Что проверяем |
|---|-------------|----------|--------------|
| 1 | **Baseline** | Чистый YOLOv12, без контекста | Нижняя граница |
| 2 | **Late Fusion** | YOLOv12 + контекст в классификаторе (как в тезисах) | Помогает ли контекст вообще? |
| 3 | **SE-Neck** | SE-Net блоки в neck | Хватает ли самоатенции каналов? |
| 4 | **CBAM-Neck** | CBAM блоки в neck | Channel+Spatial самоатенция |
| 5 | **CGFM** | FiLM-модуляция от внешнего энкодера | Внешний контекст > самоатенция? |
| 6 | **CGFM + Late** | Контекст и в neck, и в классификаторе | Максимальный эффект |

### Ожидаемые результаты

- **1 → 2:** Accuracy классификации растёт, но Precision/Recall боксов не меняется (детектор тот же)
- **1 → 3,4:** mAP немного растёт (самоатенция помогает, но ограниченно)
- **1 → 5:** Precision растёт (меньше FP), Recall растёт (меньше пропусков) → mAP растёт заметно
- **5 > 3,4:** Внешний контекст информативнее самоатенции
- **6 > 5:** Комбинация даёт лучший результат (контекст на двух этапах)

### Дополнительные аблации

**Где вставлять CGFM:** только P5 / только P3 / все уровни
**Какой энкодер:** MobileNetV3-Small (~1.5M, ~2мс) / EfficientNet-B0 (~5.3M, ~4мс) / ViT-Tiny (~5.7M, ~6мс)

## Визуализации для диплома

1. **Карта γ-значений** — тепловая карта: какие каналы усиливаются для разных типов снимков
2. **t-SNE контекстных эмбеддингов** — кластеризация по условиям съёмки (модель выучила сама)
3. **Сравнение предсказаний** — один и тот же лист: без CGFM → FP, с CGFM → правильно
4. **Гистограмма γ по классам болезней** — разные болезни активируют разные каналы

## Происхождение механизма

FiLM (Feature-wise Linear Modulation) придуман в 2018, Mila (Монреаль), Perez et al. Исходная задача — VQA: текстовый вопрос генерирует γ, β → модулирует CNN, анализирующую картинку. Потом подхвачен в генерации (StyleGAN/AdaIN), RL, аудио-визуальных задачах. В object detection neck — не переносился. Наш вклад: перенос в neck YOLO с адаптацией для аграрного контекста.
