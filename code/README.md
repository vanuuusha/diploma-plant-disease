# Код проекта — Диагностика заболеваний растений

## Структура

```
code/
├── configs/       # Конфиги экспериментов (гиперпараметры, пути)
├── data/          # Скрипты подготовки данных, data.yaml
├── models/        # Определения моделей, веса (.gitignore)
├── scripts/       # Скрипты обучения, тестирования, инференса
├── notebooks/     # Jupyter-ноутбуки для анализа
├── results/       # Результаты экспериментов (метрики, графики)
├── tasks/         # Задачи для Claude Code (TASK.md / RESULT.md)
└── docs/          # Техническая документация
```

## Система задач

Задачи создаются в `tasks/task_NN/TASK.md` и выполняются Claude Code на сервере.
Результат записывается в `tasks/task_NN/RESULT.md`.

Подробнее: см. `../CLAUDE.md` → раздел «Код и эксперименты».

## Зависимости

```bash
pip install -r requirements.txt
```
