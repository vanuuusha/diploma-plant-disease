## Цель

Проверить, можно ли поднять test-mAP YOLOv12m на `dataset_final` (aug_diffusion) за счёт подбора гиперпараметров обучения. Baseline (task_07) — mAP@50 = 0.373 (без аугментаций, дефолтные Ultralytics hyperparams). Цель — превысить 0.38 mAP@50.

## Ожидаемые хорошие кандидаты

| # | Конфиг | Что пробуем | Ожидание |
|---|---|---|---|
| 1 | **hsv_flip** | Вернуть hsv (0.015/0.7/0.4) + fliplr=0.5 (без mosaic/mixup) | Мягкие light/cam аугментации vs жёсткий diffusion-сет |
| 2 | **adamw_cos** | AdamW + cos_lr + lr0=0.001 | Лучше для малых датасетов |
| 3 | **dropout** | dropout=0.1 | Регуляризация против переобучения |
| 4 | **big_batch** | batch=48 + lr0=0.02 (scaled) | Лучший gradient estimate на A100 |

## Шаги

1. Написать runner `code/scripts/task20_hp_sweep.sh` с 4 прогонами.
2. Запустить на A100 (~1h каждый). Если кандидат даёт результат > baseline, записать в RESULT.md.
3. Выбрать лучший, документировать.

## Ожидаемый результат

- Хотя бы один прогон ≥ 0.38 mAP@50 на test.
- Лучший HP-набор для дальнейших экспериментов.
