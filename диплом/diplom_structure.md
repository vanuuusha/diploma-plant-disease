## Глава 1. Обзор методов глубокого обучения для диагностики заболеваний растений
> Расширенный НИР1 — литературный обзор актуальных подходов

### 1.1 Архитектуры для классификации изображений
- CNN: ResNet, EfficientNet, ConvNeXt
- Vision Transformers: ViT, Swin Transformer, DeiT
- Гибридные модели

### 1.2 Архитектуры детекции объектов
- Двухэтапные: Faster R-CNN, Cascade R-CNN
- Одноэтапные: **YOLOv12 (Area Attention)**
- Трансформерные: DETR, RT-DETR
- Foundation models — Grounding DINO, SAM, DINOv2

### 1.3 Методы аугментации данных
> Обзор от классических до генеративных

**Классические (геометрические/фотометрические):**
- Flip, Rotate, Crop, Color Jitter, Gaussian Blur

**Mix-методы:**
- **MixUp** (Zhang et al., 2018) — линейная интерполяция двух изображений и меток
- **CutMix** (Yun et al., 2019) — вырезание прямоугольника из одного изображения и вставка в другое
- **Mosaic** (Bochkovskiy, YOLOv4, 2020) — склейка 4 изображений в одно
- **Copy-Paste** (Ghiasi et al., CVPR 2021) — копирование объектов между изображениями (по маске)

**Генеративные (современные):**
- **GAN-аугментация** — StyleGAN, CycleGAN для генерации синтетических изображений болезней
- **Diffusion-аугментация** — DDPM / Stable Diffusion / Latent Diffusion для генерации реалистичных симптомов
- **LLM-DiffAug** (2025) — LLM генерирует текстовые описания → Diffusion модель генерирует изображения для few-shot
- **Neural Style Transfer (NST)** — перенос стиля больного листа на здоровый (LeafNST, 2024)

**Специализированные для детекции:**
- **Select-Mosaic** (2024) — адаптивный Mosaic для плотных сцен с мелкими объектами
- **Albumentations** — библиотека с bbox-aware аугментациями

### 1.4 Методы внимания и контекстной модуляции
- SE-Net, CBAM, ECA-Net — самореферентное внимание (берут сигнал из тех же признаков, которые модулируют)
- **FiLM** (Perez et al., 2018) — внешнее кондиционирование через γ, β (изначально для VQA)
- Применения FiLM: VQA, генерация, RL
- **Пробел в литературе:** FiLM не применялся в neck детекторов → наша ниша

### 1.5 VLM и мультимодальные подходы в агрономии
- Grounding DINO, CLIP, Qwen2-VL
- AgriM-LLM, PotatoGPT
- Тренд «detect → explain → recommend»

## Глава 4. Модуль контекстной модуляции признаков
> Основная научная новизна — FiLM-подход в neck детектора

### 4.1 Постановка проблемы
- Детектор «слеп» к контексту сцены (масштаб, освещение, фон)
- Самореферентные методы (SE-Net, CBAM) работают в neck, но используют как вход данные из того же neck
- Late fusion работает с готовыми bbox — контекст не влияет на саму детекцию

### 4.2 Предлагаемый подход
- Метод FiLM (2018, VQA) — перенастройка признаков через внешний сигнал (γ, β)
- Перенос в neck YOLO: отдельный энкодер смотрит на всю фотографию → понимает контекст → усиливает/ослабляет карты признаков до предсказания боксов
- Контекстный энкодер: выбор архитектуры (MobileNetV3 / EfficientNet-B0 / ViT-Tiny)
- Встраивание в FPN/PAN neck на уровнях P3, P4, P5
- Обучение end-to-end

### 4.3 Аблационное исследование
- 6 конфигураций сравнения:

| # | Конфигурация | Суть |
|---|-------------|------|
| 1 | Baseline | Чистый YOLOv12, без контекста |
| 2 | Late Fusion | Контекст после детекции, в классификаторе |
| 3 | SE-Neck | Самоатенция каналов в neck (вход = признаки neck) |
| 4 | CBAM-Neck | Канальная + пространственная самоатенция (вход = признаки neck) |
| 5 | Наш подход | FiLM-модуляция от внешнего энкодера в neck |
| 6 | Наш + Late | Внешний контекст и в neck, и в классификаторе |

- Аблация по уровням: только P5 / только P3 / все
- Аблация по энкодеру

### 4.4 Результаты и анализ
- Сводная таблица конфигураций (mAP, Precision, Recall, FPS)
- Визуализация γ-карт (какие каналы усиливаются/ослабляются)
- t-SNE контекстных эмбеддингов
- Примеры предсказаний: с модулем vs без (FP/FN анализ)
